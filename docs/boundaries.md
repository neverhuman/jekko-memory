# Boundaries

jekko-memory keeps product code under `crates/` and keeps repository control files under `agent/`, `ops/`, `scripts/`, and `.github/`.

## Ownership

`agent/owner-map.json` names the owner for each stable prefix. `agent/test-map.json` maps those prefixes to deterministic proof commands. New source roots must update both maps before code lands.

## Generated Outputs

`Cargo.lock` is a lockfile zone. `agent/repo-score.json` and `agent/repo-score.md` are auditor output zones generated by `bash ops/ci/jankurai.sh`. Hand edits to generated zones must be replaced with the source command that regenerates them.
